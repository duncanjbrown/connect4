# Connect 4

Connect 4, in ClojureScript. https://duncanjbrown.github.io/connect4/.

## Running

```
clojure -m figwheel.main -b dev -r
```

### Tests

Visit `localhost:9500/figwheel-extra-mains/tests` to view the test suite.

## Building

`make clean build`

## Deploying to GitHub pages

`./deploy.sh`
